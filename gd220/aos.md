# Using AOS in your projects

Add the following links to your `head`:

1. Library JS file:
  .text-smaller[```html
  <script src="https://unpkg.com/aos@2.3.0/dist/aos.js">
  ```]
2. Library CSS file:
   .text-smaller[```html
  <link rel="stylesheet" href="https://unpkg.com/aos@2.3.0/dist/aos.css">
  ```]
